Sight Reading Trainer

This is an interactive application that evolved from the older MIDI Sight Reader. It requires java >jdk-1.6 plugin for applet support. You can watch a video HERE. The online version is temporarily down while I transfer Asymptopia.org to GitHub.com.

The concept is still the same -- random notes are scrolled according to your settings. Notes are played aloud as they pass under the vertical red centerline. This allows the user to look ahead as necessary.

The application is written in Python (Jython). It can be run stand-alone from the terminal via either the Java interpreter or the Jython interpreter. It also runs as a Java Applet in Firefox.

Sightreadingtrainer is maintained by asymptopia.
